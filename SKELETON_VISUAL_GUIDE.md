## Color Palette

### Skeleton Colors
- **Base:** `#E5E7EB` (gray-200) at 80% opacity
- **Shimmer:** White gradient with 60% opacity
- **Borders:** `#F3F4F6` (gray-100)
- **Background:** White (`#FFFFFF`)

### Quote Card Gradients
- **Dashboard:** `from-forest/5 to-gold/5`
- **Projects:** `from-energy-green/10 to-forest/10`
- **Bills:** `from-gold/10 to-energy-green/10`
- **Settings:** `from-forest/5 to-gold/5`

## Spacing Guide

### Padding
- **Cards:** `p-6` (24px)
- **Compact cards:** `p-4` (16px)
- **Large cards:** `p-8` (32px)

### Gaps
- **Small:** `gap-2` (8px)
- **Medium:** `gap-4` (16px)
- **Large:** `gap-6` (24px)

### Margins
- **Between sections:** `space-y-6` (24px)
- **Between cards:** `space-y-4` (16px)
- **Between elements:** `space-y-2` (8px)

## Size Reference

### Heights
- **Small text:** `h-3` (12px)
- **Regular text:** `h-4` (16px)
- **Large text:** `h-5` (20px)
- **Heading:** `h-6` to `h-8` (24-32px)
- **Button:** `h-9` to `h-11` (36-44px)
- **Avatar small:** `h-10 w-10` (40px)
- **Avatar medium:** `h-12 w-12` (48px)
- **Avatar large:** `h-20 w-20` (80px)
- **Image:** `h-48` to `h-64` (192-256px)

### Widths
- **Short:** `w-16` to `w-24` (64-96px)
- **Medium:** `w-32` to `w-48` (128-192px)
- **Long:** `w-64` to `w-96` (256-384px)
- **Percentage:** `w-1/2`, `w-3/4`, `w-full`

## Border Radius

- **Small:** `rounded` (4px)
- **Medium:** `rounded-lg` (8px)
- **Large:** `rounded-xl` (12px)
- **Extra large:** `rounded-2xl` (16px)
- **Circle:** `rounded-full`

## Summary

This visual guide shows exactly how each skeleton component is structured, sized, and animated. Use it as a reference when creating new skeletons or customizing existing ones.

**Key Principles:**
1. Match the actual content layout
2. Use appropriate animation for content type
3. Maintain consistent spacing
4. Follow responsive patterns
5. Keep visual hierarchy clear
